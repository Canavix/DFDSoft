## 📌 Descripción
**DFDSoft** es una Aplicación Web Progresiva (PWA) interactiva diseñada para la creación, edición y simulación de Diagramas de Flujo de Datos (DFD). Desarrollada con tecnologías modernas (React + Vite), permite a los estudiantes y profesionales de la programación estructurar la lógica de sus algoritmos de manera visual, dinámica y altamente responsiva.

## 🚀 Características Principales
- **Interfaz Limpia e Intuitiva:** Arrastra, haz zoom y desplázate libremente por el lienzo (Canvas) de trabajo.
- **Auto-Encuadre Dinámico:** Un completo motor gráfico (Flexbox/Grid) que reajusta inteligentemente la simetría de los bloques y evita que las líneas o ramas condicionales gigantes se crucen o solapen entre sí.
- **Simulación en Tiempo Real:** Ejecuta paso a paso tus diagramas, pide valores de entrada y muestra resultados de salida usando variables en un entorno controlado.
- **Guardado y Exportación Continua:** Guarda tus proyectos en archivos locales `.dfd` y expórtalos limpiamente a imágenes `PNG` en alta definición enfocado únicamente al algoritmo, sin bordes muertos.
- **Aplicación Web Progresiva (PWA):** Instala DFDSoft en tu ordenador (Windows/Mac) o en tu celular como si fuese una aplicación nativa.

## 📖 Guía Rápida de Uso
1. **El Espacio de Trabajo:** El lienzo funciona como un mapa infinito. Arrastra con clic izquierdo para moverte y usa `CTRL + Rueda del Ratón` para acercar o alejar (Zoom). Usa el botón "Centrar" ubicado en la esquina inferior izquierda del diseñador para volver a la vista global de la aplicación.
2. **Crear Variables:** Define las variables que vas a utilizar (p. ej. `x`, `contador`, `resultado`) en el panel lateral izquierdo antes de utilizarlas en tu flujograma.
3. **Insertar Bloques lógicos:** Haz clic en el símbolo **`+`** flotante intermedio a los conectores para insertar un nuevo paso de proceso, o dentro del camino que decidas si el superior es un Condicional.
4. **Editar Bloques:** Haz **doble clic** sobre un bloque de Entrada, Salida, Condicional u Operación, para entrelazar las variables creadas con operaciones u órdenes.
5. **Ejecutar:** Presiona **Play** (botón verde superior derecho) para inicializar el algoritmo. El flujo será guiado.
